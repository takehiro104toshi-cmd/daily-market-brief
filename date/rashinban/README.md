# data/rashinban/ — 羅針盤 学習ソース置き場（v2.6）

このフォルダに岡三「羅針盤」のテキスト（**.md または .txt**）を置くと、
毎朝のGitHub Actions実行時に自動で読み込まれ、レポートの分析品質向上に
**分析フレーム（判断の型）として**利用されます。

- 本文がレポートへ転載されることはありません（ファイル名・日付・抽出フレーム数のみ表示）。
- ファイルが1つも無くても、レポートは従来どおり正常に生成されます。
- この README は読み込み対象外です（説明ファイルのため）。

## 使い方（PC・Claude Code不要。スマホのブラウザだけでOK）

### 方法A: latest.md を毎回上書きする（いちばん簡単・おすすめ）

1. GitHubでこのリポジトリを開く
2. `data/rashinban/latest.md` を開き、右上の鉛筆マーク（Edit）を押す
   - まだ無い場合は `data/rashinban/` フォルダで「Add file → Create new file」を選び、
     ファイル名に `latest.md` と入力する
3. 羅針盤の本文（テキスト）を貼り付ける
4. 「Commit changes」を押す

→ 翌朝のレポートから自動で反映されます。`latest.md` は常に最新扱いです。

### 方法B: 日付つきファイルを追加していく

`2026-07-05_rashinban.md` のように **ファイル名に YYYY-MM-DD** を入れて追加します。
日付の新しい順に最大3件（config.yamlの `rashinban.max_files`）が読み込まれます。

## 対応形式

- 対応: `.md` / `.txt`（UTF-8）
- 非対応: PDF・Word（DOCX）→ 本文をコピーして md/txt に貼り付けてください

## 何に使われるか

読み込んだテキストから、ルールベース（AI API不使用）で以下の「型」だけを抽出します。

- 相場を見る型（日本株・米国株・金利・為替の見方）
- テーマ選定・物色の型
- 注目銘柄の選び方（バリュエーション・レーティングの見方）
- リスクの見方
- 時間軸（短期/中期/長期）の置き方
- 既存 macro_themes と一致する「重点テーマ」

重点テーマに一致するニュース・テーマには、News Ranking／Strategist View／
Future Intelligence／Investment Thesis／Executive Summary で小さな補助シグナル
（+1加点・参照した旨の一文）が付きます。

## 注意

- 社内資料の本文は公開HTMLに載りません（載せない設計です）が、
  このフォルダ自体はリポジトリにコミットされます。リポジトリが
  Publicの場合は取り扱いにご注意ください（Private推奨）。
